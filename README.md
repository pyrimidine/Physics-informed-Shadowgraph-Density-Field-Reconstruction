This repository contains the source code for the research paper *"Physics-Informed Shadowgraph Density Field Reconstruction"*.

### Overview
The code implements a physics-informed framework for reconstructing density fields from shadowgraph images. The approach combines shadowgraph imaging techniques with physics-informed neural networks (PINNs) to accurately capture refractive index variations in complex flow fields.

[Animation of Flow Field Reconstruction on Alcohol Burner](./alcohol_burner_flame.gif)

### Key Features
- **Shadowgraph Image Processing**: Code for pre-processing and analyzing shadowgraph images.
- **PINN Implementation**: Physics-informed neural network setup for density field reconstruction.
- **Density Field Reconstruction**: Algorithms for computing density distributions based on refractive index data.
